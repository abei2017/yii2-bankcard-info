安装步骤
------------

你可以通过composer直接进行安装

执行

```
php composer.phar require --prefer-dist abei2017/yii2-bankcard-info // or composer require --prefer-dist abei2017/yii2-bankcard-info

```

或者在composer.json中添加

```
"abei2017/yii2-bankcard-info"
```

然后执行 composer install
